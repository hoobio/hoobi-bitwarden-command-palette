# Privacy Policy

**Command Palette Extension for Bitwarden**

*Last updated: March 6, 2026*

## Data Collection

This extension does **not** collect, store, or transmit any personal data or telemetry.

## How It Works

The extension communicates exclusively with the locally installed Bitwarden CLI on your device to retrieve vault items. All data remains on your machine and is never sent to any external server by this extension.

## Network Access

The `internetClient` capability is declared solely because the Bitwarden CLI may require network access to sync your vault. This extension itself does not make any network requests.

## Third-Party Services

This extension relies on the [Bitwarden CLI](https://bitwarden.com/help/cli/), which is governed by [Bitwarden's own privacy policy](https://bitwarden.com/privacy/).

## Contact

If you have questions about this privacy policy, please open an issue at [github.com/hoobio/hoobi-bitwarden-command-palette](https://github.com/hoobio/hoobi-bitwarden-command-palette/issues).
